# Professional Prompt-Building Framework

Use this framework to turn a visual idea into a production-ready prompt.

## 1. Objective

State what the design must accomplish.

```text
Create a campaign visual that introduces [PRODUCT] as [POSITIONING].
The viewer should feel [EMOTION] and understand [MESSAGE] immediately.
```

## 2. Core idea

Describe one strong concept instead of listing unrelated effects.

```text
The product behaves like a transparent architectural object that bends light
and reorganizes the surrounding typography.
```

## 3. Composition

Define:

- Format and aspect ratio
- Subject placement
- Camera angle
- Foreground, middle ground, and background
- Negative space
- Primary focal point

## 4. Visual system

Define:

- Color palette
- Typography character
- Materials
- Graphic language
- Grid
- Brand behavior

## 5. Lighting and realism

Define:

- Key light direction
- Shadow softness
- Reflection logic
- Skin and material texture
- Environmental integration

## 6. Quality controls

Use constraints that address common failures:

```text
No waxy skin, no malformed hands, no duplicated fingers, no floating objects,
no random interface panels, no fake depth, no excessive glow, no unreadable
typography, no inconsistent reflections, and no copy-pasted subject appearance.
```

## 7. Review loop

After each generation, assess:

1. Is the idea visible without explanation?
2. Is there one dominant focal point?
3. Does the typography belong to the composition?
4. Are anatomy and perspective believable?
5. Does the result feel designed rather than decorated?
6. What is the single weakest element?
7. Rewrite only the weak element and regenerate.
