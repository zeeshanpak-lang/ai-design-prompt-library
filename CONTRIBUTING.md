# Contributing

Thank you for helping improve the AI Design Prompt Library.

## Ways to contribute

- Add a new prompt framework
- Improve an existing prompt
- Add quality-control guidance
- Fix grammar, links, or formatting
- Add a model-specific note
- Suggest a new visual direction

## Before submitting

Please make sure your contribution:

1. Is original or appropriately attributed.
2. Does not expose private client information.
3. Does not imitate a living designer’s distinctive style by name.
4. Explains creative intent, composition, and quality controls.
5. Is useful beyond one highly specific image.
6. Uses clear Markdown formatting.

## Pull request format

Include:

- What you changed
- Why it improves the library
- Which file is affected
- Any limitations or model-specific behavior

## Prompt template

```text
## Prompt title

Purpose:
[What this prompt is designed to achieve]

Prompt:
[Reusable prompt with bracketed placeholders]

Quality controls:
[Common failure modes to prevent]
```

## Conduct

Be respectful, specific, and constructive. See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
