# AI Design Prompt Library

A curated, open-source collection of professional prompt frameworks for designers, art directors, and creative technologists working with generative AI.

The library focuses on **creative direction, visual systems, realism, composition, typography, and production quality** rather than one-click generic prompts.

## What is included

- Editorial advertising prompts
- Cinematic product and campaign prompts
- Swiss graphic design prompts
- Liquid and transparent-material prompts
- Natural portrait realism guidance
- A reusable prompt-building framework

## Why this project exists

AI can execute design rules, but strong results still require human taste, direction, references, and judgment. This project helps creators translate an idea into clear visual instructions while keeping the creative decision-making human-led.

## How to use a prompt

1. Choose a creative direction.
2. Replace the bracketed placeholders.
3. Add your brand, subject, format, and campaign goal.
4. Generate an initial version.
5. Review composition, anatomy, typography, lighting, and brand consistency.
6. Rewrite the weak parts instead of repeatedly generating the same prompt.

## Example framework

```text
Create a [FORMAT] campaign visual for [BRAND/PRODUCT].

Creative direction:
[VISUAL IDEA AND EMOTIONAL INTENT]

Composition:
[SUBJECT, CAMERA ANGLE, DEPTH, NEGATIVE SPACE]

Art direction:
[TYPOGRAPHY, MATERIALS, COLOR SYSTEM, GRAPHIC ELEMENTS]

Lighting:
[LIGHT SOURCE, CONTRAST, SHADOW QUALITY]

Quality controls:
Natural materials, intentional hierarchy, realistic anatomy, coherent reflections,
accurate perspective, no generic stock-image styling, no plastic skin, no malformed hands.
```

Contributions are welcome. Read [CONTRIBUTING.md](CONTRIBUTING.md) before opening a pull request.

Useful contributions include:

- New prompt directions
- Better quality-control language
- Before-and-after examples
- Model-specific adjustments
- Grammar and documentation improvements

## Project principles

- Human direction comes first.
- Prompts should explain intent, not only aesthetics.
- Avoid copying living designers or active brand campaigns too closely.
- Do not upload private client material.
- Do not submit prompts intended for deception, impersonation, or harmful content.
