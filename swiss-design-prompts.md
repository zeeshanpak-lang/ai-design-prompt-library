# Swiss Graphic Design Prompts

## 1. Modular Information Poster

```text
Create a Swiss-influenced poster for [EVENT/PRODUCT].

Use a strict asymmetric grid, large numerical typography, one dominant image,
and a limited palette of [COLORS]. Build hierarchy through scale, spacing, and
alignment rather than effects.

Include:
- [PRIMARY HEADLINE]
- [DATE OR NUMBER]
- [SHORT DESCRIPTION]
- [LOCATION OR CTA]

Typography must remain flat, sharp, and readable. Avoid fake 3D, unnecessary
gradients, random stickers, and decorative interface panels.
```

## 2. Typographic Motion System

```text
Design a sequence of four connected campaign frames for [BRAND].

Use one grid and one typographic system across all frames. In each frame, shift
the scale, crop, and position of [KEYWORD] to create rhythm while retaining
recognition.

The system should feel repeatable across posters, social posts, motion, and web.
Use disciplined spacing and intentional image cropping.
```

## 3. Data-Led Brand Poster

```text
Create a Swiss editorial poster that turns [DATA POINT] into the main visual idea.

Represent the number through typography and proportion rather than a conventional
chart. Add only the minimum context needed to understand it.

Use a clear grid, strong contrast, compact annotation, and a restrained color
system. The result should communicate before it decorates.
```
